== Description ==
Business Responsive is a full featured WordPress theme which includes immersive featured images functionality with a focus on business sites, that have multiple features on the front page as well as widgets, top header social icons, and navigation, a logo and more. Personalize its asymmetrical gird with a custom color scheme and showcase your multimedia content with post formats.

Business Responsive simple, straightforward typography is readable on a wide variety of screen sizes, and suitable for multiple languages. We designed it using a mobile-first approach, meaning your content takes center-stage, regardless of whether your visitors arrive by smartphone, tablet, laptop, or desktop computer.

* Responsive Layout
* Custom Colors
* Social Links

For more information about Business Responsive please go to http://bangalorethemes.com/themes/business-responsive/

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Business in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Go to http://bangalorethemes.com/themes/business-responsive/ for a guide on how to customize this theme.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.

== Home Page Setup ==
1. Goto Pages => Add New => Give it a title.
2. Select template option to set Home Page from right side page attributes.
3. Click on Update button visible top of page attributes settings.
4. Goto Settings => Reading => Select Static Page OPtion => then select a page to show your Home Page.
5. Click on Save Changes Button.

And then refresh your website you can see your all sections including(Slider,Services,Blog and Contact section) here.
